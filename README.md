## Requirements

 1. Cmake 3.8.0+
 2. CUDA 9.0+
 3. GCC 7+

## Compile && Run

```bash
git clone https://github.com/whitelok/gpu-computation-gems-codes.git
cd gpu-computation-gems-codes
mkdir build
cmake ..
make -j8
./bin/[apps you want to run]
```
